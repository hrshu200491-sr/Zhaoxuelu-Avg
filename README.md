# 照雪录 / 临安春信

一款基于 Ren'Py 制作的古风乙女向视觉小说。

故事从江南少女许知微被召入临安书院开始。她原以为自己只是来学习礼乐与绣艺，却在入院后被选入太后寿礼《临安春信图》小组。若能在年终考核前完成任务，她便有机会获得女官资格，正式留在临安；若失败，就必须回乡接受家中安排好的婚事。

临安繁华、规矩森严，也藏着意想不到的温柔。少女将在春雨、书院、宫宴、灯会与十二令风物中，遇见不同的人，也重新选择自己的人生。

## 游戏类型

- 古风乙女向视觉小说
- 女性主角
- 多男主路线
- 剧情向 / 成长向 / 恋爱向
- Ren'Py 引擎制作

## 主要角色

### 沈照雪

克制温柔型少年将军。  
行事稳重，习惯先替别人想好退路。看似冷淡，实则温柔藏在行动里。

### 萧承砚

直球护短型清冷病弱世家公子。  
身体病弱，性情清冷，却在偏爱之事上坦荡直接。

### 裴观澜

温润而有分寸的临安贵胄。  
言语得体，处事周全，温和之下藏着对世情规则的清醒判断。

### 谢兰舟

自由散漫的江湖医者。  
不拘规矩，带着药草气和雨后山风般的松弛感。

## 运行方式

本项目使用 Ren'Py 8.5.3 制作。

1. 下载并安装 Ren'Py SDK
2. 将本项目放入 Ren'Py SDK 目录
3. 在 Ren'Py Launcher 中选择项目 `zhaoxuelu`
4. 点击 Launch Project 运行

或在命令行中运行：

```bash
renpy.exe ./zhaoxuelu
```
